## 💡 Value Proposition
This project was born to bridge the gap between **aspiration** and **execution**. 
* **Grit:** Fosters perseverance through goal hierarchies (Top, Mid, and Low-level goals).
* **Antifragility:** Instead of avoiding chaos, Rafael helps users leverage obstacles as fuel for growth.
* **AI Mentor:** An integrated service powered by the Google Gemini API ("Rafael") that provides tactical and emotional feedback based on actual user performance.

## 🛠️ Tech Stack
* **Backend:** Laravel 11.x (PHP 8.3)
* **Frontend:** Vue.js 3 + Inertia.js + Tailwind CSS (VILT Stack)
* **Database:** MySQL
* **AI Integration:** Google Gemini API 
* **Infrastructure & Real-Time:** Redis (Horizon for Queues), Laravel Reverb (WebSockets), Laravel Sail (Docker), VPS Deployment (Coolify).
* **Architecture:** MVC combined with Service Layer Pattern & DDD Lite concepts.

## 🏛️ Architecture & Technical Highlights
The code snippets provided in this showcase demonstrate advanced software engineering skills:

### 1. Service Layer Pattern & Separation of Concerns
Heavy business logic (like Agenda generation or weekly analysis) is extracted into dedicated **Services** and single-purpose **Actions**. Controllers remain thin, focusing only on HTTP request handling, while Form Requests and Policies handle validation and authorization.

### 2. Resilient Artificial Intelligence (`RafaelAi.php`)
Implementation of an AI service featuring:
- **Strict JSON Protocol:** Ensures the AI always returns structured data that the backend can process seamlessly.
- **Context Injection:** The AI knows the user's profile, history, and "life goals" before providing weekly feedback.

### 3. Real-Time Communication
Implemented WebSockets via **Laravel Reverb** to push real-time notifications and enable user-to-user interactions (friends sharing goals) without relying on third-party paid services.

### 4. Standardized Timezone Handling
Database timestamps are strictly stored in UTC, while client-side views dynamically adapt to the user's local timezone using custom Traits and specific "wall-clock time" columns to ensure consistency across the globe.
